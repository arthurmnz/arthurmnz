# Arthur MNZ - Website

Frontend simples em React + Vite com Tailwind CSS, estruturado com componentes para facilitar próximas implementações.

## Stack

- React
- Vite
- Tailwind CSS

## Estrutura atual

- `src/components/PageLayout.jsx`: layout base da página
- `src/components/ProfileSection.jsx`: seção principal com o nome
- `src/App.jsx`: composição da página com componentes

## Como rodar

```bash
npm install
npm run dev
```

## Build de produção

```bash
npm run build
npm run preview
```

## Preparação para GitHub

O projeto já está com `.gitignore` e estrutura pronta para versionamento.

```bash
git init
git add .
git commit -m "feat: initial personal website"
git branch -M main
git remote add origin <URL_DO_SEU_REPOSITORIO>
git push -u origin main
```
